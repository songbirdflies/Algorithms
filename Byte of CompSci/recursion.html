Recursion & iSum

Recursive functions call itself within its own implementation. This can be done when the problem can be solved by being broken down to smaller versions of the same solution until it hits the base case. Base case is when the problem becomes so small, we can get some return value from it. It's much easier to explain with an example.

rSum(1) = 1                  => 1  **BASE CASE**
rSum(2) = rSum(1) + 2        => 3
rSum(3) = rSum(2) + 3        => 6
rSum(4) = rSum(3) + 4        => 10
rSum(5) = rSum(4) + 5        => 15

Go ahead and implement a function iSum that behaves just like rSum but instead of using recursion to implement the solution it uses iteration.
